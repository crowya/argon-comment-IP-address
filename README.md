# 鸦鸦的插件系列
- 适用于Argon主题，其他主题未经测试。
- 每个文件夹是一个插件。
# 插件功能预览
可前往[鸦鸦的巢穴](https://crowya.com/)查看效果。
## IP显示（yaya-comment-ip）
